Web Security

How would you advise a potential client to secure his/her website from web security attacks? Note: the word count for your answer should not be greater than 250 words

In order to protect a website from web security attacks you should:

  • Sanitize the website's code in order to remove unsafe content before presenting them to the user

  • Secure authorization and authentication by ensuring that they are hard to crack.

  • Use well-known third-party application that is tested and trusted

  • Continuous Deployment/Continuous Integration which catches oversights from developers and helps maintain the reliability of the changes coming to a codebase.

  • Use well known and trusted network and hosting platforms

  • Implement automatic user timeout to protect users who have left their credentials hanging in the browser for a long time.

  • Check for potential security threats on websites like this
